    设置文件永久有效链接

    MinIO 自带的文件分享不仅分享连接长而且还有时间限制。

    永久有效链接相当于文件公开，因此我们只需要将存储桶设置为公开即可实现获取文件永久有效链接。

    在 Web 控制台中修改存储桶的 Policy 为 Public。

    修改完权限后即可访问，访问链接就是 http://{MinIO地址}:{端口}/{存储桶}/{文件名}

    这里的端口是 MinIO 的 API 端口而不是 Web 控制台的端口

    如下图所示，存储位置在 test 存储桶，图片名称叫 img.png。

    那么访问链接就是 http://192.168.73.129:9000/test/img.png
